# Tech Stack

## Frontend
- **Framework**: Flutter (Desktop: Windows, macOS, Linux).
- **State Management**: Riverpod (for predictable, safe state handling).
- **UI Architecture**: Custom widgets, unified theming, `window_manager` for custom title bars, `desktop_drop` for file import.

## Backend/Processing Core
- **Language**: C/C++ or native libraries.
- **Integration**: Dart FFI for direct native performance.
- **Libraries**: Native image processing libraries (e.g., libjpeg-turbo, libwebp).
