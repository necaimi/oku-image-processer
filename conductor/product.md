# Product Definition: Oku Image Processor

## Objective
To build a high-performance, visually stunning desktop application for batch image processing (compression, resizing, format conversion). 

## Target Audience
Designers, photographers, and developers who need a fast, reliable tool with a modern UI to process multiple images efficiently.

## Core Features
1. **Batch Processing**: Resize, compress, and convert formats (PNG, JPG, WEBP).
2. **Native Performance**: Utilizing a C/C++ core via Dart FFI for blazing-fast image processing.
3. **Drag & Drop**: Seamlessly import files and folders.
4. **Real-time Preview**: Before and after comparisons of image quality and size.
5. **Custom Presets**: Save and load user-defined processing configurations.

## Design Philosophy
**Sleek Dark Minimalist**:
- Deep grays and blacks.
- Clean, geometric lines.
- Subtle glow effects for interactions.
- Custom, unified title bar integrating seamlessly with the application body.
